# Система управления студентами

Консольное приложение на Java для управления списком студентов с возможностью сортировки, поиска и анализа успеваемости.

## Возможности

- Добавление студентов с именем, возрастом и оценкой
- Отображение всего списка студентов
- Сортировка по оценке (по возрастанию)
- Сортировка по возрасту (по возрастанию)
- Поиск студента по имени
- Расчёт средней оценки всех студентов
- Определение лучшего студента (с максимальной оценкой)


### Классы

#### `students` (модель данных)

Хранит информацию о студенте:

| Поле | Тип | Описание |
|------|-----|----------|
| `name` | `String` | Имя студента |
| `age` | `int` | Возраст |
| `grade` | `double` | Оценка/балл |

**Методы:**
- Конструктор `students(name, age, grade)`
- Геттеры: `getName()`, `getAge()`, `getGrade()`
- `toString()` — форматированный вывод студента

#### `student_manager` (бизнес-логика)

Управляет коллекцией студентов:

| Метод | Описание |
|-------|----------|
| `add(students s)` | Добавление студента в список |
| `printAll()` | Вывод всех студентов |
| `sort_grade()` | Сортировка по оценке (пузырьком) |
| `sort_age()` | Сортировка по возрасту (пузырьком) |
| `findByName()` | Поиск студента по имени |
| `GetAverage()` | Расчёт средней оценки |
| `getBest()` | Поиск студента с максимальной оценкой |

## Меню программы

| Пункт | Действие |
|-------|----------|
| 1 | Показать всех студентов |
| 2 | Сортировать по оценке |
| 3 | Сортировать по возрасту |
| 4 | Найти по имени |
| 5 | Средняя оценка студентов |
| 6 | Лучший студент |
| 0 | Выйти |
